Class Location

A Location represents one service center in the real world. It can have multiple lines and service clerks, and multiple iPads and TVs. A location can have SMS enabled, when using the Pro plan.

constructor

  • new Location(properties: number | Location): Location
    • Defined in model/Location.ts:36

    Creates a new Location object, filling in all of the provided fields. You can also copy another Location by passing it to the constructor.

    example

    const location = new Location({ id: 12345, name: 'Service Hub' }); console.log(location.id); // 12345 console.log(location.name); // 'Service Hub'
