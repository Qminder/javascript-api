Class GraphQLService

A service that lets the user query Qminder API via GraphQL statements.

Note: the GraphQL API is accessible via Qminder.graphql. You should use that, instead of trying to import GraphQLService.

Static query

  • query(query: string, variables?: object): Promise<GraphqlResponse>
    • Defined in services/GraphQLService.ts:42

    Query Qminder API with GraphQL.

    Send a GraphQL query to the Qminder API.

    When the query contains variables, make sure to fill them all in the second parameter.

    For example:

    import * as Qminder from 'qminder-api';
Qminder.setKey('API_KEY_HERE');
// 1. Figure out the selected location ID of the current user, with async/await
try {
    const response = await Qminder.graphql(`{ me { selectedLocation } }`);
    console.log(response.me.selectedLocation); // "12345"
} catch (error) {
    console.log(error);
}
// 2. Figure out the selected location ID of the current user, with promises
Qminder.graphql("{ me { selectedLocation } }").then(function(response) {
    console.log(response.me.selectedLocation);
}, function(error) {
    console.log(error);
});
    throws

    when the 'query' argument is undefined or an empty string

    Parameters

    • query: string

      required: the query to send, for example "{ me { selectedLocation } }"

    • Optional variables: object

      optional: additional variables for the query, if variables were used

      • [key: string]: any

    Returns Promise<GraphqlResponse>

    a promise that resolves to the query's results, or rejects if the query failed
