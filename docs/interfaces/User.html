Interface User

Represents an employee. A User can log in to the Qminder Dashboard, manage (create/edit/call) visitors, and potentially more if they have additional rights.

Employees can have a profile picture, which will be scaled to potentially multiple sizes. The 'medium' picture size is 200x200.
